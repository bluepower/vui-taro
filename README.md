# vui-taro

VUI (based on vue3) is able to run in cross-platform solutions by Taro.

- miniprogram (weapp/swan/alipay/tt/qq/jd)
- quickapp
- h5
- react native

### Convert VUI from vui-vc-next to vui-taro

- change tag in template
  - `div` => `view`

- change event in template
  - `@click` => `@tap`
